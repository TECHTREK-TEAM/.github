## 주요 기능 
### 인증 및 사용자 관리 
* 소셜 로그인 지원: Google, Kakao, Naver OAuth를 통해 쉽고 안전하게 로그인 가능
* JWT 기반 인증 및 권한 처리: 사용자 인증과 권한을 안전하게 관리하며, 면접 데이터 접근을 보호
* 사용자 정보 조회: 계정 정보 확인, 이력서 조회/수정 등 사용자 기본 정보 관리
</br>

### 기술 면접 진행
* 기본 CS 질문, 이력서 기반 질문, 꼬리 질문 선택 가능: 기업별 정적 데이터와 GPT 기반 동적 질문으로 다양한 면접 상황을 재현
* 면접 진행 관리: 질문 제시, 답변 입력, 소요 시간 기록 등 면접 흐름을 체계적으로 관리하며 실전 대비 가능
</br>

### 답변 분석 및 평가
* 임베딩 기반 분석: OpenAI GPT 임베딩 API를 활용하여 사용자 답변과 정답 예시를 의미 벡터로 변환하고 유사도 계산
* 정확도 및 피드백 제공: 유사도 0.6 이상이면 정답 처리, 답변 일치율, 놓친 키워드 분석, 평균 점수 70 이상이면 합격으로 판단
* 실전 면접과 유사한 평가 기준: 점수화된 분석 결과와 피드백을 통해 약점과 강점을 확인하고 전략적 학습 가능
</br>

### 분석 페이지
* 이전 면접 기록 조회: 사용자가 진행한 면접 내역과 답변을 다시 확인 가능
* 답변 및 분석 결과 재확인: 면접 종료 후에도 답변과 평가 결과를 살펴보며 지속적인 학습 가능
</br>
